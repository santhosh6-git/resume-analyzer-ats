# ATS-Style Resume Analyzer

A simple web application that analyzes how well a resume matches a job description using NLP and ATS-style skill matching.

## Features
- Upload resume in PDF format
- NLP-based similarity scoring
- ATS-style weighted skill matching
- Shows matched and missing skills
- Clean dashboard with analysis history

## Tech Stack
- Python
- Flask
- Scikit-learn
- PDFPlumber
- HTML & CSS

## How to Run
```bash
pip install -r requirements.txt
python app.py
